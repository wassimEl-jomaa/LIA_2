# LIA_2
AI-assistent för testning och kvalitetssäkring som stöttar testare och testledare genom intelligent automatisering. Lösningen genererar testfall från krav, identifierar riskområden, föreslår relevanta regressionstester vid kodändringar samt sammanfattar testresultat och buggrapporter för bättre överblick och högre kvalitet.
